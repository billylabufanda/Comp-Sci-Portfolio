What is HTML?

HTML, or HyperText Markup Language, is the most common language for creating webpages. It is used to create content, and is used with CSS and JavaScript.

What are tags?

Tags are angle brackets used in HTML. They tell the computer what kind of content to render. They start with the angle brackets with the tag identifier in the middle. To indicate a tag's propperties are ending, the same tag is stated, but with a / before the tag indentifier.There are multiple examples of this:

  • The "body" tag us used to indicate the starting of displayed content. Everything you want to show should be within this tag.
  • The "h1" tag is the identifier for a heading. For example, the title at the top of this page uses a h1 tag. Smaller heading, such as the one above this section, are indicated with using a larger number. This section uses a h3 tag.
  • The "p" tag is used to contain a large amount of text that is not important enough to be a header.
  • The "ul" tag is used to start off an unordered list. For example, the "ul" tag is currently being used to start off this list of different tags.

What is CSS?

CSS, or Cascading Style Sheets, is the programming laguage for how the content is displayed. It changes the font, color, etc of the content, as well as providing a way to change the surroundings of the text, such as creating a border. To change the content, one must first indicate what one is trying to change by typing out that tag or identifier. Then, in curly brackets, one can write the stylings. There are many things one can do with CSS

  • Background-color changes the background color of the text.
  • Font-family changes the font to the font specified.
  • Border creates a border.
  • Margin creates a margin between the entity and others. For example, by putting a margin between this text box and others, there is space between them.

What are Attributes?

Attributes are ways of modifying content without using CSS. One can think of it as in-HTML CSS. It is within the tag, and start off with "style="*insert attributes here*"". An example of this would be "<*insert tag here* style="background-color:black">". This would modify the bakcground of the div to have a black background. There are other parts of the content that can be altered:

  • You can change the font using font-family.
  • You can change the color of the text using color.
  • You can change the font size using font-size.
  • You can change the background color with background-color
